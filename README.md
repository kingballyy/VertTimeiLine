**VertiLine Editor** is a responsive, interactive vertical timeline application built with **React**, **TypeScript**, and **Tailwind CSS**. It is designed to visualize historical events, project roadmaps, or any chronological data with a hierarchical structure.

### âœ¨ Features

*   **Vertical Visualization**: A clean, responsive vertical timeline layout that connects events chronologically.
*   **Hierarchical Events**: Support for nested sub-events (e.g., eras within a dynasty, battles within a war).
*   **Rich Content Editing**:
    *   **Markdown Support**: Write detailed descriptions using Markdown.
    *   **Image Attachments**: Upload and display images for each event.
    *   **External Links**: Link events to external websites for reference.
*   **Interactive Management**: intuitive drag-and-drop (planned) or click-to-edit interface to add, update, and delete events.
*   **Data Persistence**: Import and Export functionality using JSON files to save and share your timelines.
*   **Modern Tech Stack**: Built using React 19, Tailwind CSS for styling, and Lucide React for iconography.

### ğŸ›  Tech Stack
*   React 19
*   TypeScript
*   Tailwind CSS
*   Vite (Recommended for build)
